MinimalTodoApp 使用指南（v1.1.3）

一款简洁、常驻桌面的待办应用：分组管理、子待办、截止时间与倒计时、周期提醒、
日程/日历、任务置顶、Markdown 导入导出、多主题、贴边隐藏、系统托盘常驻。
界面支持中文 / English 切换。

==================================================
一、任务
--------------------------------------------------
• 在右下角输入框输入文字，回车或点「添加」即可新建任务。
• 输入文字后上方会弹出：优先级（低/中/高）、截止时间、「周期提醒」开关。
• 截止时间可选常用快捷项（5 分钟后…1 周后），也可用「自定义」精确到分钟。
  自定义选好日期/时分即自动设定，无需再点按钮；点「清除」可撤销。
• 单击任务标题可直接改文字；右键任务可：编辑、清除截止日期、缩进/提升、
  置顶、切换提醒、移动到分组、标记完成、删除。
• 任务右侧显示距截止时间的倒计时（逾期红、临近橙）。
• 勾选完成的任务自动移入「已完成」分组，取消勾选可还原。
• 拖动任务可自定义排序。

二、子待办与缩进
--------------------------------------------------
• 选中任务按 Tab 缩进成为上一条的子待办，Shift+Tab 提升一级；最多 6 级。
• 也可在任务上左右拖动改变层级；父任务左侧出现折叠箭头，点击可展开/收起。
• 通过底部「父待办」下拉给某任务添加子待办，添加后父任务自动展开。

三、任务置顶
--------------------------------------------------
• 右键任务选「置顶 / 取消置顶」。置顶的（顶层）任务会连同其所有子待办整族
  始终排在列表最上方，无论当前排序方式；置顶项带图钉标记。

四、日程 / 日历
--------------------------------------------------
• 点内容区右上角的日历按钮，在右侧展开 / 收起内嵌的日程面板（首次启动默认隐藏）。
• 可切换 天 / 周 / 月 视图：
  - 天、周视图左侧带 0–24 小时刻度轴，任务按截止时间纵向定位，一屏看完 24 小时；
    没有具体时间（00:00）的任务归到顶部「全天 / 未定时」带区。
  - 月视图按日期网格展示，任务块显示 HH:mm 时间。
• 点日历中的任务块可查看并编辑；拖动分隔条可调整面板宽度。

五、周期提醒
--------------------------------------------------
• 新建任务时勾选「周期提醒」并设置间隔（默认 30 分钟）。
  自定义提醒：填好数值、选好单位即自动设为提醒，无需再点按钮；点「清除」可撤销。
• 到点弹出左下角通知；在「设置 → 周期提醒声音」开启时会播放轻提示音。
• 任务完成后自动停止；右键任务可随时切换该开关。

六、语音输入
--------------------------------------------------
• 新任务输入框右侧的麦克风按钮触发 Windows 自带语音听写（等价 Win+H）。
• 首次使用需在「设置 → 时间和语言 → 语音」中启用语音输入。

七、导入 / 导出 Markdown
--------------------------------------------------
• 侧边栏「导入导出」可把当前清单导出为 .md，或从 .md 导入任务。
• 格式与主流编辑器兼容：分组用 ## 二级标题，任务行用 - [ ] / - [x]；
  子待办按每级两个空格缩进，导入时自动还原层级。

八、分组
--------------------------------------------------
• 左侧分组栏：点「全部任务」看全部，点分组名切换。
• 「新建分组」添加自定义分组；拖动分组可调整顺序。
• 右键分组可：修改颜色 / 清空 / 删除（「已完成」不可删除）。
• 侧边栏可拖宽；点「收起侧边栏」折叠为窄条图标，再点展开。

九、主题与设置
--------------------------------------------------
• 「修改主题」可切换内置主题（含透明、毛玻璃及多套低饱和主题）或新建自定义主题。
• 标题栏菜单 →「设置」可开关：开机自启动、完成特效、完成音效、周期提醒声音。
• 「设置」里可调整任务与正文文字的字体、字号与行距（行距同时影响文字行高与任务行间距），
  改动即时生效并自动保存；图标等界面元素不受影响。
• 点标题右侧的排序按钮选择排序方式（自定义/截止/优先级/完成/创建/标题）。
• 标题栏菜单按钮左侧的图钉按钮 = 应用置顶（窗口始终最前）。

十、贴边自动隐藏
--------------------------------------------------
• 把窗口拖到屏幕上 / 左 / 右边，窗口滑出并自动隐藏。
• 把鼠标移到对应屏幕边缘窗口自动滑回；移开再次收起。
• 再次拖动标题栏可取消贴边，恢复普通窗口。

十一、窗口与托盘
--------------------------------------------------
• 右上角红 / 黄 / 绿：关闭到托盘 / 最小化 / 最大化还原。
• 关闭仅隐藏到系统托盘、后台常驻；双击托盘图标恢复，右键托盘可退出。
